The Fading Pages plugin for ASUS Framework is designed to be used as a full-page AJAX control. More than one Fading Pages system cannot be used on one page by design. This is to keep the browser history intact and allow for hash linking directly.

Follow the example's code for structure. Note, all elements can be placed anywhere in your code and positioned / styled in any way without affecting the script. Other than .fp-nav requiring an anchor tag, all other elements can be any block-level element
